# Code of Conduct

## Our Commitment

Cooper Station is an open engineering project. We are committed to providing a welcoming and productive environment for everyone, regardless of background, experience level, or field of expertise.

## Expected Behavior

- Be respectful and constructive in all interactions
- Focus on technical merit — critique ideas, not people
- Show your work: cite sources, use SI units, document assumptions
- Welcome newcomers and help them get oriented
- Accept constructive feedback gracefully
- Respect differing viewpoints and experiences

## Unacceptable Behavior

- Personal attacks, insults, or derogatory comments
- Publishing others' private information without permission
- Sustained disruption of discussions or reviews
- Any conduct that would be considered inappropriate in a professional engineering environment

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other communication channels associated with Cooper Station.

## Enforcement

Instances of unacceptable behavior may be reported by opening a GitHub Issue or contacting the project maintainers directly. All reports will be reviewed and addressed promptly. Maintainers have the right to remove, edit, or reject contributions that violate this code of conduct.
